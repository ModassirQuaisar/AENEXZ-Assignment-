# Python Assignment

This repository contains assignments, and the following tools and technologies are primarily utilized:
- Programming Languages: Basic Python
- Environment: Jupyter Notebook.

 ## Topics Covered 

 - Foundational Programming Concepts.
   - Input/Output & Basic Operations: Calculating area based on user-defined parameters like radius.
   - Arithmetic Operations: Working with divisibility (checking for numbers divisible by 5 and 11).
     
  - Control Flow & Conditional Logic.
    - Comparison Logic: Finding the largest among three numbers.
  - Boolean Conditions:
    - Determining if a year is a leap year.
    - Checking if a character is a vowel or a consonant.
    - Identifying even numbers within a collection.
  - Prime Number Logic:
    - Checking if a specific number is prime and generating a range of prime numbers (1 to 100).
  - Iterative Statements (Loops)
    - While Loops:
      - Calculating the sum of the first $N$ natural numbers.
      - Generating multiplication tables for specific numbers.
  - Calculating the sum of the first $N$ natural numbers.Generating multiplication tables for specific numbers.
  - Nested Loops (Pattern Printing):
    - Generating pyramid structures using asterisks.
    - Printing sequential number patterns.
  - Functions & Modular Programming
    - Functional Design: Writing functions that accept parameters and return computed values (e.g., area of a circle or the largest of three numbers).
    - String Manipulation: Implementing logic to find the length of a string manually.
    - Recursion:
      - Calculating the $n^{th}$ Fibonacci number.
      - Reversing a string through recursive calls.
